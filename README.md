# Notification Service

## Overview

A React-based Notification Service application developed as part of the AffordMed Campus Evaluation.

## Features

* View Notifications
* Placement Notifications Filter
* Result Notifications Filter
* Event Notifications Filter
* Pagination Support
* Logging Middleware Integration
* Responsive UI

## Tech Stack

* React
* Vite
* Material UI
* JavaScript

## Installation

```bash
npm install
npm run dev
```

## Project Structure

* src/api
* src/components
* src/hooks
* src/pages
* src/utils
